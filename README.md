# CALCULADORA API 

Calculadora API feita em Python com Flask e integrada ao MongoDB

<br>

## Como rodar:

### Crie e ative o ambiente virtual (Linux):

~~~
$ virtualenv env
$ source env/bin/activate
~~~

### Instale as bibliotecas:

~~~
$ pip install -r requirements.txt
~~~
